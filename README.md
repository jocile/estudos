# Estudos de Formação Técnica

Repositório de notas de estudos organizadas por áreas técnicas. Os conteúdos são organizados em pastas por disciplina e mantidos atualizados conforme avanço nos estudos.

## Conteúdos Incluídos

### **Inteligência Artificial (IA)**

- Fundamentos de algoritmos de aprendizado de máquina
- Teoria de grafos e otimização
- Introdução à teoria da computação
- Modelagem de dados e estruturas
- Lógica e teoremas matemáticos aplicados

### **Web Design**

- Design responsivo e acessibilidade
- Frameworks CSS modernos (Flexbox, Grid)
- Introdução à UX/UI design
- Princípios de interface visual
- Análise de usabilidade

### **Ciências da Computação**

- Paradigmas de programação
- Teoria dos modelos computacionais
- Matemática discreta e teoria dos conjuntos
- Algoritmos e complexidade
- Sistemas operacionais

### **Programação**

- Lógica de programação
- Estruturas de dados
- Introdução a linguagens de programação
- Princípios de POO (Programação Orientada a Objetos)
- Integração de APIs

### **Redes de Computadores**

- Fundamentos de TCP/IP
- Segurança de redes
- Topologias e protocolos
- Roteamento e encaminhamento
- Comunicação de dados

### **Eletroeletrônica**

- Eletrônica analógica e digital
- Circuitos elétricos e suas leis
- Fundamentos de automação
- Sistemas de controle
- Princípios de eletricidade

## Observações

- Os arquivos são mantidos em pastas separadas por disciplina
- Sempre atualizada com novos conceitos e exemplos práticos
- Inclui diagramas e esquemas quando disponíveis

## Sobre como este repositório é construído

- Este repositório utiliza [Docusaurus](https://docusaurus.io/) para gerar uma documentação estática e organizada.
- O conteúdo é gerenciado em Markdown pelo [Obsidian](https://help.obsidian.md/) e convertido para o formato Docusaurus usando [Obsidiosaurus](https://cimsta.github.io/obsidiosaurus-docs/).

### Comandos para executar localmente

Neste diretório, podem ser executados os seguintes comandos:

  `cd website`
    Entrar na pasta do website.

  `npm start`
    Iniciar o servidor de desenvolvimento.

  `npm run build`
    Empacota seu website em arquivos estáticos para produção.

  `npm run serve`
    Serve o website construído localmente.

  `npm run deploy`
    Publica o website no GitHub Pages.

Recomendamos que você comece digitando:

  `cd website`
  `npm start`

Fechando o terminal, o servidor de desenvolvimento será encerrado automaticamente, ou você pode pressionar `CTRL + C` para encerrá-lo manualmente.:
